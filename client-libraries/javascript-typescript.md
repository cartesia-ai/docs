# JavaScript/TypeScript

{% hint style="info" %}
1. **We currently only support streaming audio to the browser.** Support for server-side environments like Bun, Node.js, and Deno is planned.
2. **The library currently only supports the WebSocket endpoint.** We will be adding support for voice cloning, server-sent events and other endpoints soon.
{% endhint %}

We offer a TypeScript library that you can use in web projects. This is the same library that powers our playground.
