## Text-to-speech minimal example

Here is a minimal example of how to use the Python client to generate a text-to-audio sample and play it.

```python
import os
import sounddevice as sd
import numpy as np
import time
from cartesia.tts import CartesiaTTS

# example
voice = 'Barbershop Man'
transcript = "Hello! Welcome to Cartesia"

# config
api_key = os.environ.get("CARTESIA_API_KEY")
assert api_key is not None
gen_cfg = dict(model_id="upbeat-moon", data_rtype='array', output_format='fp32')

# create client
client = CartesiaTTS(api_key=api_key)
voices = client.get_voices()
voice_id = voices[voice]["id"]
voice = client.get_voice_embedding(voice_id=voice_id)

# generate audio
output = client.generate(transcript=transcript, voice=voice, stream=False, **gen_cfg)

# play audio
buffer = output["audio"]
rate = output["sampling_rate"]
sd.play(buffer, rate, blocking=True)
```

Save this code snippet to `example.py` and run it with `CARTESIA_API_KEY=<enter_your_api_key> python example.py`

### Websocket connection

By default the `CartesiaTTS` client uses an HTTP POST request to dispatch the text-to-speech audio generation.

To instead use a WebSocket, wrap the call to `generate` in the `client` context manager like so:

```python
# websocket context manager
with client:
    # generate audio
    output = client.generate(transcript=transcript, voice=voice, stream=False, **gen_cfg)
```

### Output format

You can change the output format of the text-to-speech endpoint by modifying the `output_format` key in`gen_cfg`.&#x20;

The output format defines the numerical representation of the data.

```python
gen_cfg['output_format'] = 'fp32'
```

The list of valid output formats is defined in the `AudioOutputFormat` enum class.

As of writing, the supported output formats are:

* `fp32` for a 32-bit floating-point encoded audio with the default sampling rate (44.1kHz)
* `pcm` for a 16-bit signed integer PCM encoded audio with the default sampling rate (44.1kHz)
* `fp32_16000`, `fp32_22050`, `fp32_44100`, for a 32-bit floating-point encoded audio with a sampling rate of 16kHz, 22.05kHz and 44.1kHz respectively.
* `pcm_16000`, `pcm_22050`, `pcm_44100`, for a 16-bit integer encoded audio with s sampling rate of 16kHz, 22.05kHz and 44.1kHz respectively.

### Data return type

You can change the data return type of the text-to-speech endpoint by modifying the `data_rtype` key in`gen_cfg`.&#x20;

The data return type defines how the audio signal should be collated.

```python
gen_cfg['data_rtype'] = 'array'
```

The list of valid data return types is defined in the `AudioDataReturnType` enum class.

As of writing, the supported data return types are:

* `array` to return the audio as a numpy array. The audio is the list of all numbers in the signal.
* `bytes` to return the audio as a Python byte-string. The audio is the byte-string concatenation of all numbers in the signal in their Python byte-string representation.

## Text-to-speech with streaming

To enable streaming, use the following code snippet:

```python
import asyncio
import queue
import os

import numpy as np
import sounddevice as sd
from cartesia.tts import AsyncCartesiaTTS

# example
voice = 'Barbershop Man'
transcript = "Hello! Welcome to Cartesia"

# client
client = AsyncCartesiaTTS(api_key=os.environ.get("CARTESIA_API_KEY"))
voices = client.get_voices()

# config
gen_cfg = dict(model_id="upbeat-moon", chunk_time=0.1, data_rtype='array', output_format='fp32')
play_cfg = dict(channels=1)

# queue on which we push audio chunks generated by the input stream to be processed by the output stream
q_audio = asyncio.Queue()

# event that signals that the input stream finished generating the audio
done_gen = asyncio.Event()
# event that signals that the output stream finished playing the audio
done_play = asyncio.Event()

async def generate_audio_async(voice: str, transcript: str, **kwargs):
    """Input stream: Generates audio asynchronously."""
    # get voice
    voice_id = voices[voice]["id"]
    voice = client.get_voice_embedding(voice_id=voice_id)

    # async audio generator
    chunk_generator = await client.generate(transcript=transcript, voice=voice, stream=True, **kwargs)
    i = 0
    # feed chunks to input queue asynchronously
    async for chunk in chunk_generator:
        if i == 0:
            # push first chunk to initialize audio stream with appropriate parameters
            await q_audio.put(chunk)
        q_audio.put_nowait(chunk)
        i += 1
    # signal that the audio generation is done
    done_gen.set()

async def play_audio_async(channels: int, **kwargs):
    """Output stream: Plays audio asynchronously.""" 

    # feed data from q_audio to audio stream
    def callback(outdata, frames, time, status):
        if q_audio.empty():
            # audio generation has finished
            if done_gen.is_set():
                # close audio stream
                done_play.set()
                return sd.CallbackStop

            # otherwise no new data has arrived
            outdata[:] = 0
            return

        # pop last chunk
        chunk = q_audio.get_nowait()
        data = chunk['audio'].astype(dtype).reshape(-1, channels)
        assert(len(outdata) >= len(data))
        outdata[:len(data)] = data

    # create audio stream
    init_chunk = await q_audio.get()
    block_size = len(init_chunk['audio'])
    dtype = init_chunk['audio'].dtype
    sampling_rate = init_chunk['sampling_rate']
    stream = sd.OutputStream(blocksize=block_size, samplerate=sampling_rate, callback=callback, channels=channels, dtype=dtype, **kwargs)

    # play audio asynchronously 
    with stream:
        await done_play.wait()

async def main():
    # run input and output streams
    async with client:
        await asyncio.gather(generate_audio_async(voice, transcript, **gen_cfg), play_audio_async(**play_cfg))

# From jupyter notebook
# await main()
# From terminal
asyncio.run(main())
```

## Voice Cloning

{% hint style="warning" %}
Note that voice cloning is only available to Pro customers and above.&#x20;
{% endhint %}

### Voice cloning from audio clip

You can clone a given file `filepath` and obtain its embedding `voice` with the following command

<pre class="language-python"><code class="lang-python"><strong>voice = client.get_voice_embedding(filepath=filepath)
</strong></code></pre>

### Voice cloning from Youtube URL

You can clone a given `youtube_link` link and obtain its embedding `voice` with the following command

<pre class="language-python"><code class="lang-python"><strong>voice = client.get_voice_embedding(link=youtube_link)
</strong></code></pre>
