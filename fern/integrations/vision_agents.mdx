Built on [Stream Video](https://getstream.io/), [Vision Agents](https://visionagents.ai) is an open-source Video AI framework built from the ground up to enable developers to build low-latency voice and vision applications running on the edge. 

Cartesia is available as an official text-to-speech (TTS) plugin. Check out our ["Simple Agent"](https://github.com/GetStream/Vision-Agents/tree/main/examples/01_simple_agent_example) GitHub example or our [voice](https://visionagents.ai/introduction/voice-agents) and [video](https://visionagents.ai/introduction/video-agents) getting started guides to get started.
