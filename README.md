### Generative voice model

We provide the fastest text-to-speech generation model with a diverse voice library and instant voice cloning and design. Our model can generate 1s of audio with as low as 0.65s processing time and 135ms time to first audio.

{% hint style="info" %}
Given a transcript and a predefined voice, a text-to-speech model generates an audio clip of the voice uttering the given transcript.
{% endhint %}

{% hint style="info" %}
We currently only support English but are adding support for many more languages and plan to release a beta sometime soon !&#x20;
{% endhint %}

#### Voice library

We share a growing **library of voices** with different genders, ages, accents, personalities, ...&#x20;

They have been categorized according to their target audience: gaming, call center, narration, ASMR.

{% hint style="info" %}
The current library is only the beginning, we'll be adding more voices in the future. If you have any special requests please reach out to us!
{% endhint %}

#### Voice mixing

We offer the ability to **mix different voices** together, and personalize each voice by adding filters that amplify specific voice characteristics.

{% hint style="info" %}
We are working hard to add more control and better fidelity. We will post more about this in future updates.
{% endhint %}

#### Instant voice cloning

We are very excited to share our **instant voice cloning** feature which enables our text-to-speech model to imitate a voice captured in any given audio clip. This feature is only available as a paid functionality to help prevent misuse. To try out voice cloning, head to the playground at [play.cartesia.ai](https://play.cartesia.ai) and click on "Clone a voice".&#x20;

{% hint style="info" %}
Please be mindful to only upload the voices of the people for which you have explicit consent.
{% endhint %}
